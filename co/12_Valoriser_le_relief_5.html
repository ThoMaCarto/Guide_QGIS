🌈 Représenter le relief en couleur

Procédure : Symbologie « pseudo-couleurs à bande unique »⚓

Les nuances de gris vous donnent le cafard ?

Procédure
  1. Ouvrir les propriétés de symbologie de la couche MNT⚓

    Clique droit sur la couche > propriété de la couche, sélection de l'onglet symbologie dans la fenêtre qui s'ouvre.

  2. Type de symbologie : « pseudo couleurs à bande unique »⚓

    Votre MNT ne contient qu'une seule bande d'information : la bande 1 avec l'atitude. Au lieu de représenter cette information par un gradient de gris, nous allons la représenter sous la forme d'un dégradé coloré.

    Nous allons donc appliqué un dégradé coloré à une bande unique d'où le nom : « pseudo couleurs à bande unique ».

Écran : Symbologie Raster : Pseudo-couleur à bande unique⚓

  1. type de rendu
  2. la bande
  3. la palette de couleur
  4. Visualisation du dégradé
  1. type de rendu⚓
  2. la bande⚓

    Les images raster enregistre les données dans des bande. vous devez donc sélectionner la bande dans laquelle se trouve l'information que vous voulez représenter. Ici c'est facile, le MNT ne contient qu'une seule bande : la bande 1 avec l'altimétrie.

  3. la palette de couleur⚓

    séléctionner un dégradé⚓

    Si vous cliquez sur la petite flèche tout à droite, vous accédez à une liste de rampe colorimétrique préétablie et à la possibilité d'inverser le sens du dégradé (les bleus pour les valeurs faible et les rouges pour les valeurs élevées, par exemple).

    personnaliser le dégradé⚓

    Si vous cliquez sur la bande coloré, vous accédez aux paramètres de création de la rampe colorimétrique.

  4. Visualisation du dégradé⚓

    La colonne valeur correspond seuils d'altitude à partir duquel la couleur change. Ici par exemple, la couleur passe au vert à partir de 406 mètre d'altitude. Vous pouvez modifier manuellement ces valeurs si vous le souhaitez en double-cliquant sur le nombre écrit dans la colonne valeur.
