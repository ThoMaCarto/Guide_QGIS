Partager vos styles ! 🎁

Dans les propriétés de la couche, vous pouvez Charger ou Enregistrez un fichier de style.⚓

Il peut être utile de garder en mémoire le style d’une couche : sa symbologie, la façon dont elle est étiquetée. Cela permet de réutiliser le même style sur une autre couche ou de partager des styles avec vos collègues. QGIS peut enregistrer tous les paramètres d’affichage d’une couche dans un petit fichier de style que vous pourrez ensuite vous échanger ou charger dans une autre couche.

Procédure : Sauvegarder le style d’une couche :⚓

Procédure
  1. Cliquer sur Enregistrer le style...⚓
  2. Créer le fichier⚓

    Vous créez le fichier dans votre répertoire Style grâce au bouton … et vous lui donnez un nom explicite !

  3. Sélectionner les options à enregistrer⚓

    Vous cochez les différentes options que vous souhaitez enregistrer : a minima la symbologie et les étiquettes.

Procédure : Charger un fichier de style :⚓

Procédure
  1. Cliquer sur Charger le style...⚓
  2. Sélectionner le fichier contenant le style avec le bouton...⚓
  3. Vous cochez les options que vous souhaitez appliquer à votre couche : la symbologie ? Les étiquettes ? autre chose...⚓

Complément : 🎁 Cadeaux !⚓

Vous pouvez par exemple télécharger ci-dessous un document compressé contenant les styles de couches des standards de représentation des objets des documents d'urbanismes (PLU et PSMV) tel que définit pas la commission nationale de l'information géographique
