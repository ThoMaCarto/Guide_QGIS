8. Les requêtes 💬

Qu'est-ce qu'une requête ?⚓

Une requête est un outils qui permet d'interroger le SIG pour qu'il sélectionne des éléments particulier en fonction de critères définit.

Sélectionner des objets sur une couche qui répondent à certains critères spatiaux/par rapport à une autre couche (sélectionne moi toutes les entités situé dans le polygone “Gironde”) ou attributaires (sélectionne moi toutes les communes de plus de 2000 habitants).

Il y a deux type de requêtes :

les requêtes attributaires : ex. « QGIS séléctionnes-moi toutes les communes de plus de 10000 habitants dans la couche Commune ! »

le requête spatiales : ex. «  QGIS sélectionne tous les batiments qui chevauchent la limite de la commune de Pau ! »

Procédure : Effectuer une requête attributaire⚓

Procédure
  1. Cliquer sur l'outils « sélectionner les entités par attributs » dans la barre d'outils de sélection⚓

    1. La fenêtre de construction de la requête s'ouvre⚓

  2. Cliquez sur Champs et valeurs dans la colonne du milieu : la liste des champs de la table apparaît⚓
  3. Double-cliquez sur le champ qui vous intéresse pour le faire apparaître dans la case Expression à gauche de la fenêtre⚓
  4. Cliquez sur l'opérateur =⚓
  5. Cliquez sur le bouton Tous uniques pour voir dans la case Valeurs la liste des valeurs uniques du champ sélectionné⚓
  6. Double-cliquez sur la valeur qui vous intéresse⚓

    1. A ce stade, la case Expression doit contenir :⚓

      "CHAMP'' = 'Valeur'

  7. Cliquez sur le bouton Sélectionner des entités en bas de la fenêtre⚓
  8. Résultat

    Sélection⚓

    Les éléments correspondant à vos critères de sélection sont sélectionnés.

    Vous pouvez maintenant créer une nouvelle couche qui ne contient qu'eux :

    1. Clique droit sur la couche où vous avez effectué la sélection

    2. Exporter > Sauvegarder les entités sélectionnées sous ...

    3. Vous remplissez toutes les informations nécessaires à la création d'une nouvelle couche comme indiqué dans le chapitre 8.

Procédure : Effectuer une requête spatiales⚓

Procédure
  1. Cliquer sur l'outils « sélection par localisation » dans la barre d'outils de sélection⚓

    1. La fenêtre de construction de la requête s'ouvre⚓

  2. Sélectionnez les entités depuis : il s'agit de la couche dans laquelle sera faite la sélection⚓
  3. Où les entités : différents opérateurs sont disponibles.⚓
  4. En comparant les entités de : cette formulation obscure vous invite à choisir la couche par rapport à laquelle vous souhaitez sélectionner des entités⚓
  5. Vérifiez que Créer une nouvelle sélection soit bien l'option choisie⚓
  6. Cliquez sur Exécuter, vous pouvez ensuite fermer la fenêtre.⚓
  7. Résultat

    Sélection par localisation⚓

    Les éléments correspondant à vos critères de sélection sont sélectionnés.

    Vous pouvez maintenant créer une nouvelle couche qui ne contient qu'eux :

    1. Clique droit sur la couche où vous avez effectué la sélection

    2. Exporter > Sauvegarder les entités sélectionnées sous ...

    3. Vous remplissez toutes les informations nécessaires à la création d'une nouvelle couche comme indiqué dans le chapitre 8.

Commentaires

Remarque : Vous pouvez évidemment approfondir ce type de requêtes⚓

Les requêtes attributaire peuvent être plus complexes :

Par exemple, vous pouvez combiner plusieurs critères :

"INSEE-DEP '' = '33' AND ''POPULATION''>10000 cette requête sélectionnera dans la couche COMMUNE, les communes du département 33 dont la population est supérieur à 10000 habitants.

Complément : Pour plus de détails sur les requêtes⚓
