Un Symbole Unique pour les représenter tous ⭕ 🧙‍♂️

Écran : Fenêtre des propriétés de symbologie d'une couche⚓

La fenêtre de symbologie permet d'accéder à de nombreux paramètres de représentation.

  1. Menu de sélection du type de rendu
  2. Le sélecteur de symboles
  3. La sélection du type de symbole
  4. Les paramètres du type de symbole
  5. Les paramètres de rendu de la couche
  6. Le bouton style !
  1. Menu de sélection du type de rendu⚓

    C'est ici que vous choisissez le type de rendu.

    Quel que soit le type de géométrie, il y a quatre types de rendus : symbole unique, catégorisé, gradué et basé sur des règles. Pour les couches de points, il existe le déplacement de points, le nuage de points et les cartes de chaleurs, tandis que les couches de polygones peuvent être rendues avec les entités fusionnées, les polygones inversés et le rendu 2.5 D.

    Voir aussi
    • 9. Modifier l'apparence des couches Vectorielles 🎨
  2. Le sélecteur de symboles⚓

    C'est ici que vous allez sélectionner les différentes couches de symboles sur lesquelles vous allez travailler pour créer votre symbole final. Ici, il n'y a qu'une seule couche de symbole, mais vous pouvez créer des symbole complexes contenant plusieurs couches de symboles superposés. il suffit de cliquer sur le ➕ pour ajouter de nouvelles couche de symbole comme pour celui-ci qui contient un motif très complexe :

  3. La sélection du type de symbole⚓

    Vous voyez ici un symbole simple constitué de paramètres de remplissages et de paramètres de bordures. QGIS propose beaucoup d'autres possibilités.

    • des motifs de lignes ou de points
    • des textures à partir d'images pixel (motif image raster)
    • des remplissage en dégradés
    • des marqueurs aléatoires

    N'hésitez pas à tester plusieurs options et à les combiner pour créer des symboles originaux.

    Complément : Plus d'informations sur les types de symboles dans le manuel de QGIS⚓

    https://docs.qgis.org/3.40/fr/docs/user_manual/style_library/symbol_selector.html#symbol-selector

  4. Les paramètres du type de symbole⚓

    Ils varient en fonction du type de symbole choisie.

    Ici pour une remplissage simple, seul les paramètres de remplissage et de contour sont proposés.

  5. Les paramètres de rendu de la couche⚓

    Vous accédez ici à des paramètres de rendu généraux pour votre couche : opacité, mode de fusion, effets (ombrage, luminosité), et contrôle de l'ordre de rendu des objets de la couche.
