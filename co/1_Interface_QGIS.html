1. Se repérer dans l'interface de QGIS 🧭

Écran : Découvrir l'interface de QGIS⚓

Objectif⚓

Utiliser efficacement l'interface graphique de QGIS en maîtrisant les fonctions de navigation et les outils principaux.

  1. 1. La barre de menu⚓
  2. 2. les Barres d’Outils⚓

    Les groupes d’outils que vous utilisez le plus peuvent être présents sous forme de barre d’outils pour un accès facilité. Par exemple, la barre d’outils Projet vous permet de sauvegarder, charger, imprimer, et démarrer un nouveau projet. Vous pouvez facilement personnaliser l’interface pour n’afficher que les outils que vous utilisez le plus, en ajoutant ou retirant des barres d’outils via le menu Vue ► Barres d’outils.

    Même s’ils ne sont pas visibles dans les barres d’outils, tous vos outils sont accessibles via les menus. Par exemple, si vous supprimez la barre d’outils Projet (qui contient le bouton Enregistrer le projet), vous pouvez toujours sauvegarder votre carte en cliquant sur le menu Projet puis sur enregistrer.

  3. 3. Le panneau couches⚓

    Dans la liste des couches, vous pouvez voir à tout moment toutes les couches qui sont disponibles dans le projet.

    Dérouler les boutons à plusieurs options (en cliquant sur la petite flèche juste à côté du bouton) vous donnera les différentes fonctions de ses options.

    En survolant la couche, vous obtiendrez quelques informations de base : nom de la couche, type de géométrie, système de référence des coordonnées et le chemin complet de l’emplacement sur votre appareil.

  4. 4. Le Canevas / la vue cartographique⚓

    C’est là que la carte elle-même est affichée et où les couches sont chargées. Dans le canevas de carte, vous pouvez interagir avec les couches visibles: zoom avant / arrière, déplacer la carte, sélectionner des entités et de nombreuses autres opérations que nous verrons dans les sections suivantes.

  5. 5. Le panneau explorateur⚓

    L’Explorateur de QGIS est un panneau qui permet de naviguer aisément dans vos fichiers et bases de données. C’est la façon la plus simple d’accéder à des couches de différentes sortes pour les ouvrir sur votre bureau QGIS afin de travailler dessus.

    Vous pouvez créer des raccourcis vers des répertoires que vous utilisez régulièrement ⭐, ou vers le dossier de votre projet 📁.

    Retourner à l'écran
  6. 6. La Barre d’Etat⚓

    Affiche des informations sur la carte actuelle. Vous permet également d’ajuster l’échelle de la carte, la rotation de la carte et de voir les coordonnées du curseur de la souris sur la carte.

    ❓ La barre de recherche (en bas à gauche)

    Dans cette barre, vous pouvez accéder à presque tous les objets de QGIS: couches, entités de couches, algorithmes, signets spatiaux, etc.

    Retourner à l'écran

Attention : Vous avez effacé par mégarde un barre d’outils ou un panneau !⚓

Pour les faire réapparaitre sur l’interface :  

  • Dans la barre de menu cliquer sur l’onglet Vue > Panneaux ou Vue > Barres d’outils et cocher ou décocher la case qui vous intéresse.

  • Vous pouvez également faire un Clique-Droit dans une partie vide de la zone des barres d'outils (zone 2) pour afficher un menu avec toutes les options de panneaux et de barres d'outils
