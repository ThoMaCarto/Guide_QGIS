IGN : Institut National de l'Information Géographique et Forestière

C'est votre source principale pour recueillir les découpages géographiques officiels et les éléments topographiques : reliefs , bâtiment, hydrographie, route, etc. Tous les éléments concrets, pérenne, tangible qui apparaissent sur les cartes topographique de l'IGN.

Toutes ces données sont sous licence ouverte.

Vous pouvez les visualiser sur le géoportail :

geoportail_IGNF.url

Vous pouvez les télécharger sur les géoservices :

Geoservice_IGNF.url

Vous pouvez vous connecter à différentes couches de flux via les protocoles WMS/WMTS/WFS

geoplateforme_IGNF.url

Les bases de données les plus utiles de l'IGN⚓

Le catalogue de l'IGN est plétorique, voici quelques bases de données qui vous seront très utiles :

Elles sont souvent téléchargeable par département.

Parcellaire express : L'information cadastrale numérique, géoréférencée et continue sur l'ensemble du territoire français : https://geoservices.ign.fr/parcellaire-express-pci

BD TOPO® : La modélisation 2D et 3D du territoire et de ses infrastructures sur l'ensemble du territoire français => https://geoservices.ign.fr/bdtopo

RGE ALTI® : Le modèle numérique de terrain (MNT) maillé qui décrit le relief du territoire français à grande échelle => https://geoservices.ign.fr/rgealti

OCS GE : L’OCS GE est une base de données de référence pour la description de l’occupation du sol de l’ensemble du territoire métropolitain et des départements et régions d’outre-mer (DROM). =>https://geoservices.ign.fr/ocsge

Admin Express : Le découpage administratif du territoire français (commune, arrondissement départemental, département, région...).=> https://geoservices.ign.fr/adminexpress
