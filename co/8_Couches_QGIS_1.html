Créer une couche vectorielle

Complément : Les couches géographiques :⚓

Les données géographiques sont organisées en couches dans un SIG.

Chaque couche correspond à un fichier dans votre ordinateur, sur un serveur distant ou à une connexion à un serveur externe.

Il y a deux types principaux de couches :

  • Les couches Raster : des images avec des pixels ou l'information est enregistrée sous la forme d'une valeur numérique pour chaque pixel de la couche.
  • Les couches Vecteurs : des objets géométriques (points, ligne ou polygone) associé à des tables d'attributs.

Ce chapitre se concentre sur les couches vectorielles.

Nous allons voir comment créer une nouvelle couche sur laquelle nous pourrons dessiner des entités géographiques et leur attribuer des variables dans la table d'attribut.

Procédure : Créer une nouvelle couche Vectorielle⚓

Pour créer une nouvelle couche, nous allons dans la boite à outils des couches : le menu couches

Procédure
  1. Créer une couche⚓

    On commence par créer une nouvelle couche vectorielle : menu couche > créer une couche > nouvelle couche

    Le choix du format de fichier

    Il existe de nombreux formats de fichier pour enregistrer les couches vectorielles.

    Le plus répandu est le ESRI Shapefile (.shp) : très souvent demandé par la Maitrise d'ouvrage, c'est le format historique du leader mondial sur le marché ArcGIS de la société ESRI. Ce format présente cependant deux défauts : il ne peut contenir plus de 50 000 objets, il est constitué de 4 ou 5 fichiers différent qui ne doivent pas être séparés.

    Aujourd'hui, on privilégie de plus en plus le format géopackage (.gpkg) pour les très grosses base de données. Ces couches sont dans un seul fichier, qui peut contenir beaucoup plus d'entités.

    On utilise parfois le format géoJson (.geojson) pour de petite couches destinées à la cartographie interactive sur le web.

  2. Nommer et choisir l'emplacement où sera enregistré le fichier⚓

    Sur le formulaire qui s’affiche on commence par cliquer sur […] pour créer le fichier de la couche dans le bon dossier (le dossier travail, puisque c’est une couche que vous créez vous même).

    On lui donne un nom explicite par exemple : emprise_france.

  3. Choisir le type de géométrie⚓

    On précise bien la géométrie de cette couche : polygones, points ou polylignes.

    Une couche SIG ne peut contenir que des objets d'un même type de géométrie.

  4. Choisir le systèmes de coordonnées⚓

    On précise le système de coordonnée de cette couche (le même que celui du projet)

  5. Ajouter des champs⚓

    Si vous avez déjà une idée des informations attributaires que vous voudriez associer aux objets de votre couche, vous pouvez dès maintenant créer des champs. Chaque champ constitue une colonne dans le tableau d'attributs.

  6. Saisir un nom de champ⚓

    C'est le terme qui apparaîtra en en-tête de colonne

  7. définir le type de champs⚓

    C'est ici que vous déciderez de la nature de l'information saisie. Vous ne pourrez plus la changer : soit c'est une chaîne de caractère (du texte), un nombre entier, un nombre décimal, une date, etc.

    Exemples :

    Le nom de l'entité : un champ « NOM », de type « Texte(chaine de caractère) », de longueur 99

    La population résident dans l'entité en 2018: un champ « POP_2028 », de type (Entier)

    La date de mise à jour : un champ « MAJ), de type (date)

  8. Définir la longueur du champs⚓

    C'est le nombre de caractère maximum

  9. Ajouter à la liste des champs⚓

    Cliquer sur le bouton Ajouter à la liste des champs

    Vous pouvez créer autant de champs que nécessaire en reprenant à l'étape 5.

  10. Valider la création de la couche⚓

    Cliquer sur [OK]

  11. Résultat

    🔍 Vous avez désormais une nouvelle couche affichée dans votre panneau couche et un nouveau fichier dans votre dossier travail. Elle ne contient absolument aucun objet et aucune information puisque vous n’avez encore rien dessiné dessus. C’est comme si vous aviez créé un nouveau calque sur AI ou Photoshop.
