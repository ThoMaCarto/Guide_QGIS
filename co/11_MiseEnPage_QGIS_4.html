Comment créer des thèmes de cartes 👁️

Vous allez créer un thème de carte pour chacun de vos groupe de couche.

Procédure : Création de thèmes de cartes⚓

Procédure
  1. Cocher les couches que vous souhaitez voir apparaître dans votre première carte⚓
  2. Cliquez sur l’oeil 👁️ dans le panneau Couches et ajouter un thème et nommez-le.⚓
  3. Renouvelez l'opération pour toutes vos cartes.⚓
  4. Résultat

    Afficher un thème de carte⚓

    Une fois vos thèmes de cartes définis, il n'y a plus qu'à les cocher pour les afficher. Pour ce faire, ouvrez le menu thèmes de carte. une liste des thèmes du projet s'affiche. Yapuka !

    Modifier un thème de carte⚓

    Dans le menu thème de cartes, il y a l'option « Remplacer le thème »
