⛰️ Créer des courbes de niveau

Les courbes de niveau sont des isolignes ou isoplèthes.

Il s'agit de ligne qui rejoignent les points de même valeur (iso- : égal ; -ligne/-plèthe : ligne). Ici ce sont des isolignes d'altitudes.

Mais avec un Raster contenant la pression atmosphérique, nous dessinerions de isobar.

Avec un raster contenant la pluviométrie, nous dessinerions des isohyètes.

d'ailleurs, les courbes de niveau s'appelle en langage savant des isohypse (hypse = altitude en grec.). Les isohypses indiquent sur les cartes topographiques les points adjacents aux mêmes hauteurs (équidistantes). On parle également d'isobathe pour la bathymétrie, c'est à dire la profondeur dans les zones maritimes.

Procédure : Créer vos propres courbes de niveau⚓

Nous allons créer à partir de notre MNT un nouveau fichier de couche qui va extraire l'information d'altitude pour dessiner des lignes rejoignant les points de même altitude (isohypses ou courbes de niveau). Nous allons donc créer une nouvelle couche. Ce sera une couche vectorielle contenant des polylignes. Chaque polyligne sera associé à une valeur d'altitude dans sa table d'attribut.

Procédure
  1. Ouvrir l'outil « courbe de niveau »⚓

    L'outils coubre de n'iveau est un outil d'extraction d'information d'un raster. il se trouve donc dans le menu :

    RASTER > Extraction > « courbe de niveau »

    Attention :

    Selon la version de QGIS, il peut prendre des noms différents : « contour », « courbes de niveau »...

  2. Paramétrer les courbes à produire⚓

    Saisissez bien tous les paramètres :

    1. MNT source

    2. Bande

    3. Intervalle d'altitude entre les courbes

    4. entête de colonne contenant l'altitude de la courbe pour la table d'attribut

    5. Fichier à créer : ...

    6. Exécuter

  3. Résultat

    C'est fait⚓

    Vous avez désormais une nouvelle couche vectorielle contenant vos courbes de niveau.
