Découvrir et naviguer dans la table d'attributs

Vous avez ouvert la table d'attribut d'une couche. Nous allons maintenant voir différentes actions et informations auxquelles ont peut accéder.

La table d'attribut est un tableau.

Chaque ligne correspond à une entité géographique.

Les colonnes qu’on appelle des “champs” contiennent des types d’informations associés aux entités géographiques.  

Cela ressemble beaucoup à un tableur ou a une base de données, car c’est une base de données.

  1. La vue tabulaire⚓

    Chaque ligne correspond à une entité géographique.

    Les colonnes qu’on appelle des “champs” contiennent des types d’informations associés aux entités géographiques.  

    Exemple :

    Cette table de la couche des départements français contient 5 champs :

    • ID : c'est un identifiant unique pour nommer l'entité
    • NOM_M : c'est le nom du département en MAJUSCULE
    • NOM : C'est le nom du département avec une casse normale
    • INSEE_DEP : c'est le code du département utilisé par l'INSEE, le n° de département
    • INSEE_REG : C'est le code de la région utilisé par l'INSEE
  2. La barre de titre⚓

    Très utile car elle nous informe sur le nombre d'entités que contient la table, le nombre d'entité filtré et le nombre d'entité sélectionnées.

    • Combien y a-t-il de départements français dans cette couche ?
    • Sont-ils bien tous là ?Y a-t-il eu un flltrage ?
    • Combien sont sélectionnées ?

    ...

  4. Les filtres⚓

    Réduire la quantité d'entités affichées dans le tableau⚓

    En bas à gauche, un menu déroulant vous propose différentes fonctions pour filtrer les entités qui s'afficheront dans le tableau.

  5. Sélectionner des entités⚓

    Cliquez sur son numéro de ligne tout à gauche⚓

    Si vous cliquez sur le numéro d'une ligne, elle apparaîtra en surbrillance bleu. Cela signifie qu'elle est sélectionnée.

    Regardez sur la vue cartographique, l'entité sélectionné devrait apparaître en jaune.

    Vous pouvez sélectionner plusieurs ligne en maintenant la touce CTRL ou Maj appuyée.

    Retourner à l'écran
  6. Basculer sur la vue formulaire/tabulaire⚓

    Attention : Vous avez basculez en vue formulaire dans la table d'attribut !⚓

    Pour revenir à la vue en tableau, cliquez sur l'icone tableau en bas à droite.

    Retourner à l'écran
  7. En-têtes de colonnes pour classer⚓

    Les entêtes de colonnes permettent de classer les entités par ordre ascendant ou descendant
